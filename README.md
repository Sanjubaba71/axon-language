🚀 What Is AXON?

AXON is a working AI reasoning assistant that goes beyond normal AI answers.

Instead of giving flat responses, AXON:

- Adds confidence scores
- Generates alternative possibilities
- Uses attention-based reasoning
- Simulates memory decay
- Maintains internal states (qualia)

👉 It behaves more like a thinking system than a chatbot.

⚡ Features

- ✅ Confidence-based outputs

▶️ How to Run (Local Setup)

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

3. Enter your API key in the UI

---

🧠 Architecture

app.py          → Streamlit UI
axon_engine.py  → Reasoning pipeline
axon_core.py    → AXON runtime logic

AXON Concepts

- know(x, confidence) → store value with uncertainty
- remember(x, decay) → fading memory
- consider() → decision branching
- explore() → hypothesis generation
- attend() → attention weighting
- qualia() → internal state
- paradox() → contradiction handling
